CSS Position Property has 4 possible values

  1. Static
  2. Relative
  3. Absolute
  4. Fixed

Static

  1. Static is the default postion value.
  2. Top, bottom, left, and right will NOT work with static.
  3. A static element will show up exactly where it is in the HTML, left to right, to to bottom... it CANNOT be moved.
Static box

Relative Position

  1. A relative position is the same as static, EXECPT it will honor top, left, bottom, right
  2. If you move it, it will sit on top of other stuff on the page, and the page will think it's still where it used to be.
Relative Box

Fixed Position

  1. A fixed postion element will remain, relative to the viewpoint (the screen) even if you scroll
  2. Top, bottom, and right will work just fine
  3. Unlike Relative, it will not reserve its old space. IE., anything with static or relative will not know it's there.
Fixed Box

Absolute Position

  1. Absolute elements work off the grind system. Just like fixed, they are ignored by EVERYONE.
  2. They honor z-index
  3. Like fixed, they do not leave gaps in the page
  4. The top/left/bottom/right will be relative to the FIRST parent that has a posisiton set.
