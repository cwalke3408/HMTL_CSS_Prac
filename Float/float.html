Float (makes no sense)

  1. A block level element, when floated, longer takes up the whole parent. It's no longer a block. It's not even a display type. You can't add margin.
  2. The element, will go as far left (or right) as it possibly can, unti it runs into another floating thing or it runs out of room.
  3. A floated element is in it's own universe. They don't pass their height up to non-floating parents. Static + float = weird
  1. IMPORTANT
  2. To force a floated elemnt to pass it's height up to it's parent...
  3. Make the parent float
  4. Set the parent to have overflow: hidden
